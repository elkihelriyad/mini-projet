% ============= Chapter 1: Introduction =============
\chapter{Introduction}

\section{Contexte du Projet}

L'École Nationale des Sciences Appliquées (ENSA) de Safi propose six filières d'ingénierie distinctes aux étudiants ayant terminé leur cycle préparatoire. Le choix de la filière est une étape cruciale qui détermine le parcours professionnel futur de l'étudiant.

\begin{infobox}[Problématique]
Comment aider les étudiants du cycle préparatoire à identifier la filière qui correspond le mieux à leurs intérêts, compétences et aspirations professionnelles ?
\end{infobox}

\section{Objectifs du Projet}

Le projet \textbf{ENSAGuide} vise à :

\begin{enumerate}[leftmargin=*]
    \item Développer un \textbf{test d'orientation intelligent} basé sur un système de pondération.
    \item Créer une \textbf{interface web moderne} et responsive.
    \item Intégrer un \textbf{chatbot conversationnel} pour répondre aux questions fréquentes.
    \item Fournir des \textbf{fiches détaillées} pour chaque filière (programme, débouchés, compétences).
\end{enumerate}

\section{Système de Pondération}

Chaque question du test est associée à un \textbf{vecteur de poids} (JSON) qui indique son influence sur chaque filière.

\begin{infobox}[Exemple de Pondération]
\textbf{Question :} \textit{"J'aime résoudre des problèmes algorithmiques complexes."}\\[0.3cm]
\textbf{Poids :} \texttt{\{"GIIA": 5, "GMSI": 2, "GTR": 1\}}\\[0.3cm]
\textbf{Interprétation :} Une réponse élevée (4-5) à cette question favorisera fortement la filière GIIA.
\end{infobox}

\subsection{Formule de Calcul des Scores}

Pour chaque filière $f$, le score total est calculé comme suit :

\[
\text{Score}_f = \sum_{q \in Q} \text{Réponse}_q \times \text{Poids}_{q,f}
\]

Où :
\begin{itemize}
    \item $Q$ est l'ensemble des questions
    \item $\text{Réponse}_q \in \{1, 2, 3, 4, 5\}$ est la réponse de l'utilisateur
    \item $\text{Poids}_{q,f}$ est le poids de la question $q$ pour la filière $f$
\end{itemize}

\chapter{Module Chatbot}

\section{Technologie : TF-IDF et Similarité Cosinus}

Le chatbot utilise une approche basée sur la \textbf{recherche d'information} (Information Retrieval) pour trouver la réponse la plus pertinente dans sa base de connaissances.

\subsection{Algorithme}

\begin{enumerate}[leftmargin=*]
    \item \textbf{Vectorisation TF-IDF :} Transformation des questions de la knowledge base en vecteurs numériques.
    \item \textbf{Transformation de l'input :} La question de l'utilisateur est également vectorisée.
    \item \textbf{Calcul de similarité :} Utilisation de la \textbf{similarité cosinus} entre l'input et chaque question.
    \item \textbf{Sélection :} La réponse associée à la question ayant la plus haute similarité est retournée (si > seuil 0.3).
\end{enumerate}

\begin{codebox}[Extrait de chatbot.py]
\begin{lstlisting}[language=Python]
def get_response(self, user_input):
    user_vec = self.vectorizer.transform([user_input])
    similarities = cosine_similarity(user_vec, self.tfidf_matrix)
    best_idx = np.argmax(similarities)
    score = similarities[0][best_idx]
    
    if score > 0.3:
        return self.answers[best_idx]
    else:
        return "Je ne suis pas sur de comprendre..."
\end{lstlisting}
\end{codebox}

% ============= Chapter 6: Fonctionnalités =============
\chapter{Fonctionnalités Clés}

\section{Test d'Orientation Intelligent}

\begin{enumerate}[leftmargin=*]
    \item L'utilisateur répond à 6 questions sur une échelle de Likert (1-5).
    \item Les réponses sont envoyées via \textbf{Fetch API} au backend.
    \item Le serveur calcule les scores pondérés pour chaque filière.
    \item Les \textbf{Top 3 filières} sont affichées avec leurs scores.
\end{enumerate}

\section{Fiches Filières Détaillées}

Chaque filière dispose d'une page complète avec :
\begin{itemize}
    \item Description et objectifs
    \item Débouchés professionnels (affichés en tags)
    \item Mots-clés / compétences (affichés en hashtags)
    \item Programme de formation sur 3 ans (timeline visuelle)
\end{itemize}

\section{Chatbot Intégré}

Le widget de chat permet aux utilisateurs de poser des questions directement depuis n'importe quelle page. Exemples de questions supportées :
\begin{itemize}
    \item "C'est quoi GIIA ?"
    \item "Quels sont les débouchés de GTR ?"
    \item "Comment choisir ma filière ?"
\end{itemize}
